七、核心状态机 State Machine

1. 加速连接状态

  idle (home)  ──[点击连接]──→  connecting (进度0→100%)  ──[100%]──→  connected (计时)
       ▲                              │                                    │
       │                      [点击取消]                            [点击停止]
       └──────────────────────────────┴────────────────────────────────────┘

2. 登录状态

  未登录 (logged=false)
    ├── 短信登录：输入手机号 → 获取验证码(60s倒计时) → 输入验证码 → 登录
    └── 一键登录：显示脱敏号码(天翼认证) → 点击一键登录 → 登录
                                                         │
                                                         ▼
  已登录 (logged=true) ──[退出登录/注销]──→ 未登录

3. VIP 会员状态

  非会员 (isVip=false)
    │
    ├── [选择套餐] → [勾选协议] → [点击CTA] → Apple IAP 支付
    │                                              │
    │                                         [支付成功]
    │                                              │
    ▼                                              ▼
  会员 (isVip=true, vipExpiry='2026.12.31')
    │
    └── [到期/注销] → 非会员
    
  套餐选项：
    - id:0  1个月   ¥6(首月) / ¥30(续期)  新人价
    - id:1  6个月   ¥120  省¥60           推荐
    - id:2  年度    ¥198  省¥162          最划算 ← 默认选中

4. 注销流程

  Step 1: 用户点击 [确认注销] 按钮
              │
              ▼
  Step 2: 显示输入框，提示输入「确认注销」四个字
              │
              ├── 输入正确 → [注销账户] 按钮激活 → 点击执行注销
              │                                        │
              │                              清空状态(logged/isVip/phone)
              │                              跳转 profile
              │
              └── [取消] → 回到 Step 1

App Store 审核合规要点

必须满足 (3.1.2a 合规)：
1. 「恢复购买」入口必须可见（当前在底部订阅说明区）
2. 自动续订说明必须清晰：到期前24h自动续期 + 取消路径
3. 订阅页必须包含：付费协议、服务协议、隐私政策 三个链接
4. 协议勾选必须在「立即开通」按钮之前
5. 未勾选协议时点击 CTA 必须有明确提示（当前为抖动+高亮）

SDK 集成

  • 天翼账号认证 SDK：一键登录（本机号码识别）
  • 友盟统计 SDK：埋点与崩溃收集
  • VPN 协议库：按实际加速方案选型（如 WireGuard / Shadowsocks）

适配要求

  • 最低支持：iOS 15.0+
  • 设备适配：iPhone SE 2 ~ iPhone 16 Pro Max
  • 安全区：顶部 StatusBar 动态适配（刘海/灵动岛）、底部 Home Indicator
  • 屏幕模式：仅深色模式（不支持亮色切换）
  • 方向：仅竖屏（Portrait）

十、H5 开发专项

技术选型

项目推荐方案
框架React 18 + TypeScript
构建Vite 5
适配方案375px 设计稿，postcss-px-to-viewport 或 rem (1rem=37.5px)
路由React Router v6（若需独立路由）/ 单组件状态切换
动画CSS @keyframes（优先）+ Framer Motion（复杂交互）

性能要求

  • 所有动画使用 GPU 加速属性：transform, opacity
  • 避免触发 layout/reflow 的属性动画（width, height, top, left）
  • 星星粒子/速度线使用 will-change: transform
  • 图片资源使用 WebP 格式，SVG 图标内联

分享功能降级

  navigator.share 可用？
    ├── YES → 调用系统分享面板
    └── NO  → 降级方案：
              ├── 微信环境 → wx.js-sdk 分享接口
              └── 其他浏览器 → 复制链接到剪贴板 + Toast 提示
              
  分享内容：
    title: "星舟加速器"
    text: "星际穿梭·极速畅游"
    url: App Store 链接

兼容性

  • 浏览器：Safari 15+, Chrome 90+, WebView (WKWebView)
  • 禁止：使用 -webkit-overflow-scrolling（已废弃）
  • 滚动：scrollbar-width: none + ::-webkit-scrollbar{display:none}
